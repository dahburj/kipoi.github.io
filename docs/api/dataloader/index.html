Dataset

kipoi.data.Dataset()

An abstract class representing a Dataset.

All other datasets should subclass it. All subclasses should override __len__, that provides the size of the dataset, and __getitem__, supporting integer indexing in range from 0 to len(self) exclusive.

PreloadedDataset

PreloadedDataset(self)

Generated by supplying a function returning the full dataset.

The full dataset is a nested (list/dict) python structure of numpy arrays with the same first axis dimension.
